6. Το 400 πιστεύω είναι το κατάλληλο status code καθώς λείπει ένα υποχρεωτικό όρισμα. Ένα τέτοιο λάθος είναι σφάλμα σύνταξης που ορίζεται από την εφαρμογή. Συνεπώς είναι ένα κακό αίτημα.

7. Σίγουρα θα μπορούσε να εμφανίσει ένα status code 500 αλλά νομίζω το 507 ίσως για την συγκεκριμένη περίπτωση.

8. Το POST είναι προτιμότερο, καθώς οι λεπτομέρειες θα αποστέλλονται στο σώμα των μηνυμάτων HTTP και όχι στη διεύθυνση URL.

9. Η διαφορά μεταξύ POST και PUT είναι ότι τα αιτήματα PUT είναι ανεπαρκή. Δηλαδή, η κλήση του PUT πολλές φορές θα παράγει πάντα το ίδιο αποτέλεσμα. Αντίθετα, η επανειλημμένη κλήση ενός POST θα κάνει αλλαγές πάνω στο πρώτο αποτέλεσμα.

10. Για να «ανεβασουμε» ένα αρχείο θα πρέπει να χρησιμοποιήσουμε την μέθοδο Post.
